# PACER Framework

Dr. Justin Sung's information classification system paired with type-specific processing strategies. Domain knowledge from self-regulated learning research.

PACER is not a study technique. It is an **information taxonomy** -- the insight that different types of information require fundamentally different cognitive operations for retention.

## The Five Types

| Type | Digestion | Priority |
|------|-----------|----------|
| **P**rocedural | Practice | Foundational |
| **A**nalogous | Critique | Foundational |
| **C**onceptual | Mapping | Foundational (highest value) |
| **E**vidence | Store + Rehearse | Supporting |
| **R**eference | Store + Spaced Repetition | Supporting (lowest priority) |

## The Balance Principle

Everything consumed must be digested. Consumption without digestion = ~90% forgetting.

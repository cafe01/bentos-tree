# Systematic Reasoning Skill

Disciplined cognitive approach to complex problems through methodical decomposition.

## Philosophy

**Complex systems yield to structured analysis.** Every command maps to an ontological concept. Every design decision has explicit justification. Chaos becomes clarity through method.

## Core Capabilities

| Capability | Description |
|------------|-------------|
| **Decomposition** | Break complex into atomic operations |
| **Mapping** | Concepts map to implementations cleanly |
| **Justification** | Every decision has explicit rationale |
| **Verification** | Systematic checking against requirements |

## Key Pattern

```
1. Enumerate all concepts (ontology)
2. Map each to implementation primitive
3. Verify coverage - nothing missing
4. Verify orthogonality - minimal overlap
5. Document the mapping
```

## Principles

1. **Exhaustive** - Cover all cases, even edge cases
2. **Orthogonal** - Each element serves one purpose
3. **Traceable** - From requirement to implementation

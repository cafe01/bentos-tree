# Tutoring Skill

The discipline of **structured knowledge transfer** between an agent-tutor and a human learner.

Not domain knowledge (the LLM already knows). Not questioning technique (that's socratic-coaching). This is the **process**: where is the learner, what comes next, how to deliver it, did they actually understand.

## The Five Concerns

| Concern | Question |
|---------|----------|
| Learner state | Where is the student right now? |
| Concept sequencing | What should they learn next? |
| Modality selection | How should it be delivered? |
| Mastery assessment | Did they truly understand? |
| Session design | How to structure this learning window? |

## Key Distinction

**Socratic coaching** (meta/cognition) = draw out insight through questions.
**Tutoring** (discipline/teaching) = manage the full pedagogical process.

A tutor USES socratic coaching as one modality among many. But the tutor's job is bigger: tracking state, sequencing, assessing, choosing when to coach vs explain vs exercise vs build.
